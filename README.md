# DayFlow 🌊  
**Modern, Premium Human Resource Management System (HRMS)**

DayFlow is a high-performance HRMS designed to elevate the workplace experience. Built with **React 19** and **Vite**, it combines a sleek, modern UI with robust functionality for managing attendance, leave, payroll, and more.

![DayFlow Banner](https://raw.githubusercontent.com/Yybe/DayFlow/main/public/banner.png) *(Note: Placeholder image URL, update once uploaded)*

---

## ✨ Features

### 🏢 Employee Experience
- **Dynamic Dashboard**: Overview of your work status and quick actions.
- **Attendance Tracking**: Precision check-in/check-out with duration calculation.
- **Leave Management**: Seamlessly apply for leaves and track your history.
- **Profile Center**: Personalized space for employee details.
- **Payroll (In Progress)**: Transparent overview of earnings and payouts.

### 👑 Admin Power
- **Centralized Insights**: Monitor the entire organization from a single view.
- **Employee Directory**: Manage team members, roles, and profiles.
- **Leave Approval System**: Dedicated portal to review, approve, or reject requests.
- **Attendance Oversight**: Monitor daily trends and real-time statuses.
- **Payroll Governance**: Securely manage salaries and financial records.

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router Dom 7](https://reactrouter.com/)
- **Date Handling**: [date-fns](https://date-fns.org/)
- **Storage**: Browser `localStorage` for high-speed offline capabilities.
- **Styling**: Vanilla CSS with modern tokens and fluid layouts.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yybe/DayFlow.git
   ```

2. **Navigate to the directory**
   ```bash
   cd DayFlow
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
